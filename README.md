# Shopping Cart

A fully functional mock e-commerce shopping cart built with React. Browse products fetched from the FakeStore API, add them to your cart, adjust quantities, and remove items — all with real-time cart count updates in the navbar.

## Features

- Home page with featured products fetched from FakeStore API
- Shop page with full product grid — add items with quantity controls
- Cart page — view, update quantities, and remove items
- Live cart item count in the navbar
- Client-side routing with React Router
- Loading and error states on all data fetches
- PropTypes validation on all components
- Tested with React Testing Library and Vitest
- Styled with CSS Modules

## Built With

- React
- React Router
- Vite
- FakeStore API
- CSS Modules
- Vitest + React Testing Library
